# Akshita Maheshwari Portfolio

This is a complete responsive portfolio website for Akshita Maheshwari, Senior Software Engineer - Backend & Cloud.

## Files

- `index.html` - portfolio page content and structure
- `styles.css` - responsive styling
- `script.js` - mobile menu and year update
- `portfolio-content.md` - copy-ready portfolio text

## How to preview locally

1. Download and unzip the folder.
2. Open `index.html` in your browser.

## Before publishing

Update these items:

1. Replace the LinkedIn button URL in `index.html` with your exact LinkedIn profile URL.
2. Add your GitHub URL if you want recruiters to review code samples.
3. Add your resume PDF and link it from the hero section if desired.
4. Remove phone number if you do not want it publicly visible.

## Easy publishing options

### GitHub Pages

1. Create a new public GitHub repository, for example `akshita-portfolio`.
2. Upload these files.
3. Go to Settings > Pages.
4. Under Source, select `Deploy from a branch`.
5. Select `main` branch and `/root` folder.
6. Save and wait for GitHub to generate your portfolio URL.

### Netlify

1. Go to Netlify.
2. Drag and drop the unzipped folder.
3. Netlify will instantly create a public URL.

### Vercel

1. Create a new Vercel project.
2. Import the GitHub repository.
3. Deploy as a static website.
